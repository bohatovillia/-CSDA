## Опис проєкту

Реалізація вбудованого HTTP-сервера на мікроконтролері **ESP32** з використанням:
- Файлової системи **LittleFS** для зберігання HTML-контенту у Flash-пам'яті
- Стека **TCP/IP (lwIP)** через бібліотеку `WiFi.h` Arduino
- Симуляції у середовищі **Wokwi** (замість CY8CKIT-062 через відсутність апаратного модуля)

## Швидкий старт (Wokwi)

1. Відкрийте [wokwi.com](https://wokwi.com) → Create New Project → ESP32
2. Замініть `sketch.ino` вмістом з `sketch/sketch.ino`
3. Натисніть ▶ (Play) для запуску симуляції
4. У Serial Monitor з'явиться IP-адреса пристрою
5. Відкрийте IP у браузері (в симуляції відображається у логах)

## Технічний стек

| Компонент | Опис |
|-----------|------|
| ESP32 DevKit C V4 | Мікроконтролер з Wi-Fi/BT (замість CY8CKIT-062) |
| LittleFS | Файлова система у Flash-пам'яті |
| lwIP | Легковажний TCP/IP стек (вбудований у ESP32 Arduino) |
| HTTP Server | `WebServer.h` — обробка GET-запитів на порту 80 |
| Wokwi | Онлайн-симулятор для тестування без апаратного забезпечення |
