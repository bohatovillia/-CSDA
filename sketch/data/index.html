🖌 Embedded Web Server

ESP32 HTTP Server

Лабораторна робота — вбудований вебсервер на мікроконтролері

🔌 Статус системи

HTTP-сервер активний — порт 80
LittleFS змонтована успішно
Wi-Fi підключено (STA режим)

📄 Про проєкт

Цей вебсервер запущений безпосередньо на мікроконтролері ESP32. HTML-сторінка зберігається у Flash-пам'яті через файлову систему LittleFS та відправляється клієнту при GET-запиті на корінь /.

⚙ Технічний стек

MCUESP32 DevKit C V4
HTTP StackWebServer (lwIP)
File SystemLittleFS (Flash)
NetworkWi-Fi IEEE 802.11b/g/n
IDE / SimWokwi + Arduino
PortTCP 80

🔨 Алгоритм роботи сервера

  • Ініціалізація Wi-Fi у режимі WIFI_STA
  • Підключення до точки доступу, отримання IP через DHCP
  • Монтування LittleFS, читання /index.html
  • Реєстрація callback-обробника на маршрут /
  • Обробка HTTP GET-запитів, відправка HTML-відповіді 200 OK
